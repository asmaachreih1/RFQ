This project is a frontend implementation of a Request for Quotation (RFQ)
Marketplace built with Next.js (App Router) and TypeScript.

## Features

- User & Company roles
- RFQ creation and management
- Quotation submission and comparison
- Smart quotation sorting (price + delivery)
- Best offer highlighting
- Category subscriptions
- Real-time in-app notifications via WebSocket
- Responsive UI with modern layout

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- CSS Modules
- WebSocket (native)
- Context API for state management
